# ساختار سimple قدیمی (تاریخی)

فقط **یک جدول** `anime` — برای آرشیو نگه داشته شده.

| ستون         | نوع        | توضیح                    |
|-------------|------------|---------------------------|
| `id`        | uuid       | شناسه یکتا |
| `title`     | text       | عنوان انیمه              |
| `cover_image` | text     | لینک تصویر جلد           |
| `synopsis`  | text       | خلاصه داستان (اختیاری)  |
| `format`    | text       | TV، MOVIE (اختیاری) |
| `average_score` | number | امتیاز (اختیاری) |
| `episodes`  | integer    | تعداد قسمت‌ها (اختیاری)  |
| `studio`    | text       | استودیو (اختیاری)        |
| `season`    | text       | فصل پخش (اختیاری) |
| `start_date`| text       | تاریخ شروع (اختیاری) |
| `end_date`  | text       | تاریخ پایان (اختیاری) |
| `created_at`| timestamp  | زمان ثبت |
